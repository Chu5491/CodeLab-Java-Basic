# 추상클래스

## 개념 

- 구현 코드가 없이 선언부만 있는 **추상 메서드**를 포함한 클래스  
- `abstract` 예약어를 사용하여 클래스 정의  
- 추상 클래스는 `new` 예약어를 사용할 수 없음, 즉 인스턴스로 바로 생성할 수 없음  
- 추상 클래스는 구현 클래스를 통해서 구현한 뒤에 인스턴스로 사용할 수 있음
