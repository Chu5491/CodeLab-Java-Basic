# Ch03. 연산자(1)

## #1 단항 연산자

### [부호연산자](./Ex1_UnaryOperator.java#L6)

> **+** : 해당 값을 그대로 전달  
> **-** : 해당 값의 부호를 반대로 교환  
> 부호 연산자는 변수에 저장된 값에 영향 없이 호출된 값에만 적용됨

### [증감연산자](./Ex2_InDecrementOperator.java#L6)

> **++** : 해당 값을 1 올림  
> **--** : 해당 값을 1 내림  
> 전위 증감연산 : 해당 값을 해당 행 연산이 시작되기 전에 1 증감  
> 후위 증감연산 : 해당 값을 해당 행 연산이 끝난 다음에 1 증감
